# 關於每日 AI 新聞

本站是**每日精選摘要**，整理前一天最重要的 AI 新聞、論文與產品發布。

## 運作方式

每天上午 10:00（台北時間），自動 agent 會：

1. 掃描主要 AI 來源（arXiv、HuggingFace、OpenAI、Anthropic、Google AI、Meta AI、GitHub Trending、Hacker News 等）
2. 篩選最重要的事件
3. 撰寫雙語摘要（繁體中文 + English）
4. 自動發布到本站

## 內容架構

- **首頁**：直接顯示當日最新摘要
- **彙整頁**：列出所有歷史摘要，方便回顧
- **關於頁**：本站介紹

## 資料來源

- arXiv（cs.AI、cs.CL、cs.LG）
- 主要 AI 實驗室部落格（OpenAI、Anthropic、Google DeepMind、Meta AI、Mistral、xAI）
- Hacker News（AI 標籤且 >100 分的故事）
- GitHub Trending（AI/ML 專案）
- 主要科技媒體（TechCrunch、The Verge、Reuters）
- 第三方 AI 新聞彙整（buildfastwithai.com 等）
